# Sistema de Gestão de ONG

Projeto desenvolvido em Java para a disciplina de Programação Orientada a Objetos.

## Conceitos aplicados

* Encapsulamento
* Herança
* Sobrescrita de métodos
* Composição

## Estrutura

* MembroONG (superclasse)
* Voluntario (subclasse)
* Doador (subclasse)
* ProjetoSocial (composição)
* SistemaMain (classe de teste)
